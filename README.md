# Kotlin-ViewPager2-JsonFile-CountDownTimer
Basic Wallet App

Data Repo -> res/raw/wallet.json

WalletModel -> Data Model
WalletAdapter -> Card Number, CVC Code set to wallet (ViewPager Adapter)
MainActivity -> Read data from raw file, timer handle, json format converting, view bind
Helper -> price convert extension
